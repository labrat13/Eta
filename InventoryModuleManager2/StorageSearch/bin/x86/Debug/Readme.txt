Браузер Хранилищ
Версия 1.0 от 19 октября 2017г

Описание:  Это программа для поиска в Хранилищах типа MeramanStorage InventoryModule2.

Возможности:
- поиск по паттерну *pattern* или pattern* или *pattern в Хранилищах. 
- показывает дерево классов 
- показывает дерево мест - Хранилищ.
- открывает связанный с записью документ
- отображает связанное с записью изображение
- формирует файл - список дерева классов сущностей.
- показывает неисправные Хранилища иконкой ошибки в дереве мест.

Недостатки:
- это прототип, поэтому все криво и функционал ограничен.
